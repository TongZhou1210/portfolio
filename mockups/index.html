Assignment #1 — Yatzy Game (Front-end)

This single-player Yatzy game was developed using HTML, CSS, and JavaScript modules. It demonstrates modular programming, clean UI design, and accessibility features.

Core Modules

  • dice.js — Dice state and rolling logic.
  • yatzyEngine.js — Score calculation and rules.
  • yatzyGame.js — Game flow controller.
  • script.js — DOM events and UI rendering.

Key Features:

  • Hold / re-roll dice, 13 scoring categories, upper bonus, and total score tracking.
  • High score persistence using localStorage.
  • Accessible interface (ARIA labels and keyboard navigation).
  • Modular JavaScript architecture using ES6 import/export.
